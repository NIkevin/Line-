# 📌 LINE 貼圖裁切工具

## 功能簡介
此工具可將圖片裁切成 LINE 貼圖所需的格式，支援 **2x2、3x3、自訂比例**，並能自動生成主要圖片 (main.png) 與聊天室縮圖 (tab.png)。最後可下載符合 LINE 規範的 ZIP 壓縮檔。

---

## 🔧 操作步驟

### 1. 選擇裁切方式
- 在下方選擇欄位中，選擇裁切方式：
  - **2x2**：將圖片裁切成 4 張
  - **3x3**：將圖片裁切成 9 張
  - **自訂比例**：可輸入自訂行列數進行裁切

---

### 2. 上傳圖片
- 點擊上傳區域或拖放多張圖片
- 上傳主要圖片 (**main.png**) 與聊天室縮圖 (**tab.png**)
- 若未上傳：
  - **main.png** → 系統自動選擇第一張裁切圖片
  - **tab.png** → 系統自動選擇最後一張圖片

---

### 3. 圖片裁切與顯示
- 點擊 **「處理所有圖片」** 按鈕
- 系統會依照選擇的裁切方式，將圖片裁切並依序顯示
- 每張裁切結果會顯示：
  - 預覽圖片
  - **刪除按鈕**（移除不需要的圖片）
  - **勾選框**（確認要保留的圖片）

---

### 4. 管理裁切結果
- 可點擊 **刪除按鈕** 移除不需要的圖片
- 可透過 **勾選框**確認要保留的圖片

---

### 5. 下載結果
- 點擊 **「下載貼圖格式」**
- 系統會自動打包成 **stickers.zip**
- 壓縮檔內包含：
  - main.png
  - tab.png
  - 勾選保留的裁切圖片
